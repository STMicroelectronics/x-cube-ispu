Purpose

The X-CUBE-ISPU Projects provide examples and applications to aid understanding how to develop an application using the middlewares and drivers for the MEMS devices. Most projects include STM32CubeMX project file (.ioc) for the graphical visualization of the STM32 microcontroller pins, peripherals and middlewares configuration.
